# Recipe App

A modern React-based recipe application that allows users to explore, search, and discover delicious recipes. Browse popular and trending recipes, filter by categories, and get detailed cooking instructions.

## Features

- **Popular Recipes Slider**: Discover the most loved recipes in a sleek carousel.
- **Trending Recipes Slider**: Stay updated with the latest trending dishes.
- **Recipe Search**: Search for recipes by name or ingredients.
- **Category Filtering**: Browse recipes by categories like breakfast, lunch, dinner, etc.
- **Detailed Recipe View**: Get full recipe details including ingredients, instructions, and nutritional info.
- **Responsive Design**: Optimized for desktop and mobile devices.

## Tech Stack

- **Frontend**: React 19
- **Routing**: React Router DOM
- **UI Components**: Custom components with CSS
- **Sliders**: React Slick for smooth carousel experiences
- **Build Tool**: Create React App
- **Testing**: Jest and React Testing Library

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd recipeapp
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

- **Home Page**: View popular and trending recipes via sliders.
- **Search**: Use the search bar to find specific recipes.
- **Categories**: Click on category links to filter recipes.
- **Recipe Details**: Click on any recipe to view full details, ingredients, and instructions.

## Available Scripts

- `npm start`: Runs the app in development mode.
- `npm test`: Launches the test runner.
- `npm run build`: Builds the app for production.
- `npm run eject`: Ejects from Create React App (irreversible).
